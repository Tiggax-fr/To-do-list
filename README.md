# To-Do List

Aplicação simples de lista de tarefas feita com HTML, CSS e JavaScript puro, com visual moderno, tema claro/escuro e paleta verde como destaque principal.

## Recursos

- Adicionar novas tarefas
- Marcar tarefas como concluídas
- Remover tarefas da lista
- Filtrar entre todas, concluídas e pendentes
- Alternar entre tema claro e escuro
- Contador automático de itens pendentes e totais

## Visual

- Tema escuro com fundo profundo e detalhes em verde
- Tema claro com aparência limpa e suave
- Botões, campos e cartões harmonizados com a identidade verde

## Estrutura do projeto

- [index.html](index.html) — estrutura da aplicação
- [style.css](style.css) — estilos e temas visuais
- [script.js](script.js) — lógica da aplicação e interações

## Como executar

1. Abra a pasta do projeto no VS Code.
2. Abra o arquivo [index.html](index.html) no navegador, ou use uma extensão como Live Server.
3. Digite uma tarefa, clique em Adicionar e use os botões para gerenciar a lista.

## Comportamento da aplicação

- **Concluir** alterna o estado da tarefa entre pendente e concluída.
- **Deletar** remove a tarefa imediatamente.
- Os filtros atualizam a lista em tempo real.
- O tema escolhido é salvo no navegador.

## Tecnologias

- HTML5
- CSS3
- JavaScript (ES6+)
