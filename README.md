# To-Do List (HTML, CSS e JavaScript)

Projeto simples de lista de tarefas com interface moderna e funcionalidades essenciais em JavaScript puro.

## Funcionalidades

- Adicionar nova tarefa
- Concluir tarefa (risca o texto e reduz opacidade)
- Deletar tarefa
- Contador automático de tarefas pendentes e totais

## Estrutura do projeto

- `index.html` — estrutura da aplicação
- `style.css` — estilos visuais
- `script.js` — lógica da aplicação (CRUD básico de tarefas na interface)

## Como executar

1. Abra a pasta do projeto no VS Code.
2. Abra o arquivo `index.html` no navegador (duplo clique ou extensão Live Server).
3. Use o campo de texto para adicionar tarefas.

## Comportamento da aplicação

- Ao clicar em **Concluir**, a tarefa alterna entre concluída e pendente.
- Ao clicar em **Deletar**, a tarefa é removida da lista.
- O contador no topo é atualizado automaticamente a cada ação.

## Tecnologias

- HTML5
- CSS3
- JavaScript (ES6+)
